Barclays Credit Card Customer Retention & Advanced Lookup Tool
Project Overview
This project focuses on analyzing raw banking records of 10,000+ credit card clients to identify customer churn patterns and risk segments. It provides actionable insights for risk management and includes an automated query engine for fast data retrieval.
 Technical Implementation & Features
 Tool Used: Advanced Microsoft Excel.Data Cleaning Pipeline: Processed and structured raw banking records using conditional logic to classify churned profiles.
 Automated Query Engine: Designed a high-speed customer profile search engine utilizing combinations of VLOOKUP, XLOOKUP, and INDEX-MATCH.
 Executive Analytics Dashboard: Developed interactive summary pivot tables and customized column charts with dynamic Slicers (Filters) for real-time stakeholder review.
 Key Business Insights & ImpactHigh-Risk Segment Identified: Discovered that Platinum Card holders have the highest customer churn rate (25%), providing critical dropout insights for risk management teams.
 Efficiency Boost: Significantly reduced data fetching time for customer profiles through the automated search tool.
