Barclays Credit Card Churn Analysis

Project Overview
This project focuses on analyzing raw banking records of 10,000+ credit card clients to identify customer churn patterns. It provides actionable insights for
risk management teams.

Key Features & Tools Used
Tool Used: Advanced Microsoft Excel
Data Cleaning: Processed structured records using conditional logic to classify active vs churned profiles.
Automated Query Engine: Designed a customer profile lookup system using VLOOKUP, XLOOKUP, and INDEX-MATCH.
Executive Dashboard: Developed summary pivot tables and column charts with dynamic Slicers.

Key Business
InsightsHigh-Risk Segment: Discovered that Platinum Card holders have the highest customer churn rate (25%), highlighting a major retention issue.
Efficiency: Significantly reduced the time required to fetch individual customer profiles.
