# Barclays Credit Card Churn Analysis

## Project Overview
This project focuses on analyzing credit card customer data to identify churn patterns and customer retention risks. The analysis helps understand customer behavior, detect high-risk customer segments, and support data-driven business decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- VLOOKUP
- XLOOKUP
- INDEX-MATCH

## Key Features

### Data Cleaning
- Organized and cleaned customer records for accurate analysis.
- Standardized data fields and prepared the dataset for reporting.

### Customer Lookup System
- Developed an automated customer profile lookup system using VLOOKUP, XLOOKUP, and INDEX-MATCH.
- Improved efficiency in retrieving customer information.

### Interactive Dashboard
- Built dynamic Pivot Tables and Pivot Charts.
- Added Slicers for interactive filtering and analysis.
- Created a user-friendly dashboard for business reporting.

## Key Business Insights

### Customer Churn Analysis
- Identified customer segments with higher churn risk.
- Analyzed customer behavior and retention trends.

### High-Risk Segment
- Found that Platinum Card holders had the highest churn rate, indicating a potential retention challenge.

### Operational Efficiency
- Reduced the time required to retrieve and analyze customer information through automated lookup functions.
